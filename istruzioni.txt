*** PARTE 1: installare i requisiti ***
1) Accedere alla cartella zoraProject_SentenceClassification da terminale
2) Creare un environment e installare i requisiti, ci vorrà qualche minuto per scaricare i modelli:
	python -m venv env
	su windows:
	cd env/Scripts
	activate
	su macOs:
	cd env
	source bin/activate	
	cd ..
	cd server
	pip install -r requirements.txt

*** PARTE 2: configurazione del progetto lato server ***
1) Aprire la cartella server, inserire l'api-key di openai nel file config.py
2) Inserire l'ip del pc nella variabile hostName dentro i file server1.py, server2.py, server3.py, server4.py
3) In 4 terminali separati attivare l'environment e aprire un server per ogni terminale
	python serverX.py

*** Parte 3: configurazione del progetto lato robot ***
1) Aprire Choregraphe, selezionare import project from CRG file, selezionare il file "ZoraBotProject_SentenceClassification.crg", importare il file
2) Modificare la variabile url con l'ip inserito nei server nei seguenti blocchi 
	SpeechToText, SpeechToText(1), Decide
	AskOrMove -> AskToGpt e modificare nei blocchi AskToGPT e TalkAndMove

#### Ora il progetto è pronto a essere eseguito da Choregraphe ####
